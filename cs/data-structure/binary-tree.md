# Binary Tree

## Height

|                      | Minimum Height                | Maximum Height |
|----------------------|-------------------------------|----------------|
| Binary Tree          | $\lceil\log_2(n+1)\rceil - 1$ | $n-1$          |
| Full Binary Tree     | $\lceil\log_2(n+1)\rceil - 1$ | $(n-1)/2$      |
| Complete Binary Tree | $\lceil\log_2(n+1)\rceil - 1$ | $\log_2(n)$    |
